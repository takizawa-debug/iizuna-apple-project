飯綱町 産業振興・観光推進プロジェクト
UX/UI DESIGN

1. ビジュアルと検索の融合

飯綱町の「りんご」という強力なブランド資源を際立たせるミニマリズムと、ユーザーの知りたい情報へ即座にリーチできる統合検索システムを構築しました。

ブランディング: 現代的なタイポグラフィと高画質素材の活用
統合検索: 物理ボタンとデジタル検索のシームレスな体験
GLOBAL STANDARDS

2. 待機なしの多言語体験

インバウンド対応を強化するため、3言語（日・英・繁）の即時切り替えを実装。ページ遷移のストレスを排除したモーダル形式を採用しています。

パンフレット発行: 閲覧中の情報を即座にPDF化
SNS連携: 魅力的な情報のバイラル性を最大化
DATA DRIVEN

3. 次世代型アクセス分析

GA4の複雑さを排除し、現場が「今、何が必要か」を即断できる専用ダッシュボードを構築。前期間比較による施策検証が容易です。

ALGORITHM

4. 「真の滞在時間」の可視化

タブの切り替えやバックグラウンド状態を除外し、Page Visibility API を活用したアクティブ精読時間のみを計測。コンテンツの真の価値を浮き彫りにします。

ActiveTime = TotalTime - BackgroundHiddenTime
RELIABILITY

5. データの整合性・地域分析

セッション単位ではなくユーザー単位での地域集計に刷新。全ての地域別ランキングの合計がトータルUU数と完全に一致する高い精度を実現しました。

本資料の出力方法

Googleドキュメント/スライドへの出力は、同梱の DeliveryExporter.gs をGoogle Apps Scriptで実行してください。
